# 카카오 캐러셀 디자인 명세서

**규격**: 750 × 1000 px (캐러셀)
**최종 업데이트**: 2026-04-30

---

## 📐 캐러셀 기본 규격

### 사이즈
- **너비**: 750px
- **높이**: 1000px
- **카드 종류**: 표지(cover), 내지(inner), 아웃트로(outro)

### 폰트 (시스템)
- **폰트 패밀리**: Pretendard
- **소스**: Google Fonts CDN

### 색상 시스템 (미리보기 카드 내부)

미리보기 카드에서 사용되는 색상은 빌더 UI 시스템 토큰(--y100~y300)과 별개.
미리보기 색상은 카카오 브랜드 가이드를 따름.

| 용도 | 색상 코드 | 비고 |
|------|----------|------|
| 카카오 브랜드 노랑 | `#FEE500` | 버튼 기본값 |
| 카카오 브라운 | `#3C1E1E` | 노랑 배경 위 텍스트 |
| 강조 빨강 (할인) | `#EF3C2C` | 할인율 표시 |
| 다크 텍스트 | `#111` | 헤드라인 |
| 서브 텍스트 | `#5F5F5F` | 본문 |
| 힌트 텍스트 | `#C0C0BC` | 비활성 |
| 라이트 배경 | `#F5F5F5` | 카드 배경 |
| 화이트 배경 | `#FFFFFF` | 내부 박스 |

## 🎨 카드 C — 혜택형 (내지)

> **목적**: 다양한 혜택을 배지로 시각화하여 강조  
> **사용 위치**: 내지 (Inner)

### ⏳ 상세 명세 (작성 예정)

#### 레이아웃
- 상단 띠배너 영역:
- 이미지 영역:
- 배지 영역:
- 하단 유의사항 영역:

#### 타이포그래피
- 띠배너 텍스트:
- 배지 라벨 (할인 등):
- 배지 수치 (57%, 15% 등):
- 유의사항 텍스트: 26px / 두께 300 (Light) / 색상 rgba(255,255,255,0.7)

#### 색상
- 배지 기본 배경: `#2F2F2F` (이미지 어두울 때 `#3F3F3F`)
- 결제사 배지: 결제사별 브랜드 컬러 (`_payC` 객체)
- 무료배송 배지: `#2F2F2F`
- 사용자기재 배지: `#2F2F2F`
- 띠배너 컬러: 사용자 지정 (`bannerColor`)

#### 배지 종류
- **무결제계열**: 즉시할인, 상품쿠폰, 장바구니, 무료배송, 사용자기재
- **결제사계열**: 카카오페이, 네이버페이, 토스페이, 오늘의집페이

#### 배지 규칙
- 1개일 때: 모든 배지 가능
- 2개일 때: 1번은 무결제계열, 2번은 자유
- 3개일 때: 1~2번은 무결제계열, 3번은 결제사계열

#### 배지 간격
- 세로 배지 간 gap: 10px
- 배지 위치: 우측 상단 (top 30px 또는 134px when 띠배너 있음)
- 배지 크기: 160 × 160 px (원형)

#### 띠배너 종류
- 브랜드형: 사용자 컬러 + 텍스트
- 오마트: 고정 컬러 #ff8442
- 오세일: 사용자 컬러
- 오늘의딜: 고정 컬러 #FF4747
- 스오딜: 5탭 표시
- 없음: 띠배너 미표시

#### 인터랙션
- 이미지 위치 조정: ✅ 가능
- 자동 컬러 추출: ❌ 비활성
- 배지 밝게 토글: ✅ `darkImg` (이미지 어두울 때 배지 살짝 밝게)
- 유의사항 표시 토글: ✅ `showNotice`
- 유의사항 텍스트 편집: ✅ `noticeText` (30자 제한)
